# Data Directory

Large raw datasets and generated H2 display databases are intentionally not tracked in Git.

Expected local files include, for example:

- `frontend_template_data.mv.db`
- `ani_gdb_s03.h5`
- `ani1x-release.h5`
- `Transition1x.h5`
- `data0000.aselmdb`
- `calculated_polymer_data.xlsx`

Rebuild or restore these files locally according to the project documents before running the full database.
